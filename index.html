Finotes Integration - Quickstart

Add finotes to your Android and iOS Application to capture and report in real-time Function failures, Workflow failures, Network Failures and Crashes!. Integration typically takes < 10 mins.

  1. Sign up to fi.notes. and login
  2. "Add app" to link android and iOS application to fi.notes.
  3. Integrate fi.notes SDK to your application. Refer documentation -Android - iOS objective C
  4. Test integration. Refer documentation - Android - iOS objective C
  5. Proceed as per the documentation to monitor the issues listed here.
